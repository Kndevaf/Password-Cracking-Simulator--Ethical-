# Password Cracking Simulator (JavaScript)

An educational, browser-based JavaScript project that demonstrates **why weak passwords are insecure** by simulating common attack techniques such as dictionary attacks and brute-force checks.

 **This project is for learning and awareness only.**  
No real cracking or malicious behavior is performed.

---

## Features
- User enters a password
- Simulated **dictionary attack**
- Simulated **brute-force resistance check**
- Real-time feedback in the browser
- Clean, modern UI
- No backend or external libraries required

---

## Technologies Used
- HTML5
- CSS3
-  JavaScript (ES6)
